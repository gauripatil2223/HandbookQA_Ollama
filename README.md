### HandbookQA ###
A friendly company handbook assistant that answers your questions using Ollama + FAISS. Ask about policies, working hours, leave, or remote work — it finds the info and tells you confidently

#### How it works ####
-Handbook chunks are embedded with nomic-embed-text
-FAISS finds the most relevant sections
-Llama3 summarizes and answers your question
-Confidence scores show how reliable the answer is
-Always cites sources — never guesses
